# Mars Dashboard App 
### Description

App displaying latest photos from Mars rovers using NASA api. Project aim is to demonstrate functional programming in JavaScript. 

### How to run

1. Install the dependencies:
```yarn install``` 

2. Run the app:
```yarn start``` 

3. Access app from http://localhost:3000
